# tabnews-curso.dev

This repository is part of my learning journey in **curso.dev**.

The project focuses on rebuilding a **TabNews-like application** while applying the **core technologies and practices** required to design, build, test, and deploy a real-world web system.

The emphasis is on fundamentals that transfer across stacks, not on superficial framework usage.

---

## Goal

- Build a production-oriented web application from scratch  
- Apply real software engineering workflows  
- Practice architecture, testing, security, and APIs  
- Develop confidence to work with modern web systems  

---

## Core Technologies & Concepts

### Languages & Runtime
- **JavaScript**
- **Node.js**
- **NVM** for Node version management
- **NPM** for dependency management

### Web & APIs
- **Next.js** (full-stack framework)
- **RESTful APIs**
- HTTP methods and status codes
- Stateless application design

### Code Quality & Testing
- **Jest** for automated tests
- Unit and integration testing
- Code formatting and consistency
- Incremental refactoring

### Version Control & Collaboration
- **Git**
- **GitHub**
- Pull Requests and Code Review
- Branch-based workflow
- Issues and Milestones for tracking progress

### Database
- **PostgreSQL**
- SQL fundamentals
- Parameterized queries
- Database migrations

### Security
- Password hashing
- Authentication and authorization concepts
- Environment variables for sensitive data
- Protection against common vulnerabilities (e.g. SQL Injection)

### Infrastructure & Deployment
- Environment separation (local, staging, production)
- Continuous Integration concepts
- Production-oriented project structure
