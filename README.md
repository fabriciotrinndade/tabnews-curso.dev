# tabnews-curso.dev

Practical implementation inspired by **TabNews**, developed during **curso.dev**, focused on backend development, software architecture, automated testing, and engineering best practices.

The project prioritizes long-lasting fundamentals that remain relevant regardless of frameworks.

---

## Purpose

Build a well-structured backend application to study and consolidate core concepts such as:

- Software architecture
- HTTP APIs
- Version control best practices
- Automated testing
- Code organization for real-world projects

---

## Technologies

- **Node.js**
- **JavaScript**
- **Next.js (API Routes)**
- **Jest** (unit and integration tests)
- **PostgreSQL**
- **Prisma**
- **Docker**
- **Git & GitHub**
